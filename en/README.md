# Introduction

1. Install CLI, install npm ahead of time 
```bash
npm install gitbook-cli -g
```
2. Install plugins from book.js
```bash
gitbook install
```
3. Build
```bash
gitbook build . --log=debug
```
4. Preview
```bash
gitbook serve
```
